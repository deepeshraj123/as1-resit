CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.

In Css I have used the simple Background colors and color of texts and font family. The font family I chose is "Lucida Sans", "Lucida Sans Regular", "Lucida Grande", "Lucida Sans Unicode", Geneva, Verdana, sans-serif so that text will look more appealing. I have also used the classes in the css to make the ui of navigation bar.
